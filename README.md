# Gif's for a Queen

This is a web app that can take emotion (or any word) and pull related gifs - with a drag queen twist. This relies on the Giphy API to get the relevant gifs to display on screen.

## Technologies Used"

1. Javascript 
2. Jquery
3. AJAX
4. GIPHY API
5. Bootstrap
6. Google Fonts
7. HTML
8. CSS
